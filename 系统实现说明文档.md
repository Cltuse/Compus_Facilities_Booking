# 校园公共设施预约管理系统 - 系统实现说明文档

## 1. 系统总体架构设计

### 1.1 技术架构概述
本系统采用前后端分离的微服务架构，后端基于Spring Boot 3.2.0框架构建，前端采用现代化Web技术栈。系统设计遵循MVC模式，实现高内聚低耦合的模块化设计。

**核心技术栈：**
- **后端框架**: Spring Boot 3.2.0
- **数据库**: MySQL 8.0
- **持久层**: Spring Data JPA
- **Java版本**: Java 21
- **构建工具**: Maven
- **服务器端口**: 5681

### 1.2 系统架构分层
系统采用经典的三层架构设计：
- **表现层**: Controller层，负责接收HTTP请求和返回响应
- **业务逻辑层**: Service层，处理核心业务逻辑
- **数据访问层**: Repository层，负责数据持久化操作

### 1.3 模块化设计原则
系统按照功能模块进行划分，每个模块包含完整的MVC结构：
- **实体层**: Entity类，对应数据库表结构
- **数据访问层**: Repository接口，提供数据操作
- **业务逻辑层**: Service类，实现业务规则
- **控制层**: Controller类，处理HTTP请求

```java
@PostMapping("/register")
public Result<Map<String, Object>> register(@RequestBody User user) {
    // 1. 检查用户名是否已存在
    Optional<User> existingUser = userRepository.findByUsername(user.getUsername());
    if (existingUser.isPresent()) {
        return Result.error("用户名已存在");
    }

    // 2. 检查邮箱是否已存在
    if (user.getEmail() != null && !user.getEmail().isEmpty()) {
        Optional<User> existingEmailUser = userRepository.findByEmail(user.getEmail());
        if (existingEmailUser.isPresent()) {
            return Result.error("邮箱已被使用");
        }
    }

    // 3. 设置默认值
    if (user.getRole() == null || user.getRole().isEmpty()) {
        user.setRole("USER");
    }
    if (user.getStatus() == null || user.getStatus().isEmpty()) {
        user.setStatus("ACTIVE");
    }

    // 4. 保存用户信息
    User savedUser = userRepository.save(user);

    // 5. 返回用户信息（用于自动登录）
    Map<String, Object> data = new HashMap<>();
    data.put("id", savedUser.getId());
    data.put("username", savedUser.getUsername());
    data.put("realName", savedUser.getRealName());
    data.put("role", savedUser.getRole());
    data.put("phone", savedUser.getPhone());
    data.put("email", savedUser.getEmail());
    data.put("avatar", savedUser.getAvatar());

    return Result.success("注册成功", data);
}
```

### 4.3 预约创建流程实现

```java
@PostMapping
public Result<Reservation> create(@RequestBody Reservation reservation) {
    try {
        // 1. 基本数据验证
        if (reservation.getFacilityId() == null) {
            return Result.error("设施ID不能为空");
        }
        if (reservation.getUserId() == null) {
            return Result.error("用户ID不能为空");
        }
        if (reservation.getStartTime() == null) {
            return Result.error("开始时间不能为空");
        }
        if (reservation.getEndTime() == null) {
            return Result.error("结束时间不能为空");
        }
        
        // 2. 检查设施是否存在且可用
        Optional<Facility> facilityOpt = facilityRepository.findById(reservation.getFacilityId());
        if (!facilityOpt.isPresent()) {
            return Result.error("设施不存在");
        }
        
        Facility facility = facilityOpt.get();
        if (!"AVAILABLE".equals(facility.getStatus())) {
            return Result.error("该设施当前不可用，无法预约");
        }
        
        // 3. 检查用户是否存在
        Optional<User> userOpt = userRepository.findById(reservation.getUserId());
        if (!userOpt.isPresent()) {
            return Result.error("用户不存在");
        }
        
        // 4. 验证预约规则
        Result<String> ruleValidationResult = validateReservationRules(reservation, facility);
        if (!ruleValidationResult.isSuccess()) {
            return Result.error(ruleValidationResult.getMessage());
        }
        
        // 5. 检查时间有效性
        if (reservation.getEndTime().isBefore(reservation.getStartTime())) {
            return Result.error("结束时间不能早于开始时间");
        }
        
        // 检查预约时间范围（不能预约过去的时间）
        if (reservation.getStartTime().isBefore(LocalDateTime.now())) {
            return Result.error("不能预约过去的时间");
        }
        
        // 6. 检查时间冲突
        List<Reservation> existingReservations = reservationRepository.findByFacilityId(reservation.getFacilityId());
        for (Reservation existing : existingReservations) {
            if (isTimeConflict(reservation.getStartTime(), reservation.getEndTime(), 
                             existing.getStartTime(), existing.getEndTime()) &&
                !("REJECTED".equals(existing.getStatus()) || 
                  "CANCELLED".equals(existing.getStatus()))) {
                return Result.error("该时间段已被预约，请选择其他时间");
            }
        }
        
        // 7. 根据规则设置状态并保存
        RuleConfig ruleConfig = getApplicableRuleConfig(facility);
        String initialStatus = (ruleConfig != null && ruleConfig.getNeedApproval()) ? "PENDING" : "APPROVED";
        reservation.setStatus(initialStatus);
        reservation.setCheckinStatus("NOT_CHECKED");
        
        Reservation savedReservation = reservationRepository.save(reservation);
        enrichReservation(savedReservation);
        
        String message = "预约成功";
        if ("PENDING".equals(initialStatus)) {
            message += "，请等待管理员审核";
        }
        
        return Result.success(message, savedReservation);
    } catch (Exception e) {
        e.printStackTrace();
        return Result.error("预约失败：" + e.getMessage());
    }
}
```

```java
public String validateReservationCreation(Reservation reservation) {
    // 1. 基本数据验证
    if (reservation.getFacilityId() == null) return "设施ID不能为空";
    if (reservation.getUserId() == null) return "用户ID不能为空";
    if (reservation.getStartTime() == null) return "开始时间不能为空";
    if (reservation.getEndTime() == null) return "结束时间不能为空";
    
    // 2. 设施可用性检查
    Optional<Facility> facility = facilityRepository.findById(reservation.getFacilityId());
    if (!facility.isPresent()) return "设施不存在";
    if (!"AVAILABLE".equals(facility.get().getStatus())) return "设施不可用";
    
    // 3. 用户状态检查
    if (blacklistRepository.findByUserIdAndStatus(reservation.getUserId(), "ACTIVE").isPresent()) {
        return "用户已被列入黑名单";
    }
    
    // 4. 时间有效性检查
    if (reservation.getEndTime().isBefore(reservation.getStartTime())) {
        return "结束时间不能早于开始时间";
    }
    if (reservation.getStartTime().isBefore(LocalDateTime.now())) {
        return "不能预约过去的时间";
    }
    
    // 5. 预约时长限制
    long durationHours = Duration.between(reservation.getStartTime(), reservation.getEndTime()).toHours();
    if (durationHours > 24) return "单次预约时长不能超过24小时";
    
    // 6. 时间冲突检测
    List<Reservation> conflicts = findConflictingReservations(
        reservation.getFacilityId(), reservation.getStartTime(), reservation.getEndTime()
    );
    if (!conflicts.isEmpty()) return "该时间段已被预约";
    
    return null; // 验证通过
}
```

```java
/**
 * 验证用户签到操作
 */
public String validateCheckin(Reservation reservation, Long userId) {
    // 验证用户身份：只有预约用户本人才能签到
    if (!reservation.getUserId().equals(userId)) {
        return "只有预约用户本人才能进行签到操作";
    }
    
    // 检查预约状态
    if (!"APPROVED".equals(reservation.getStatus())) {
        return "只有已通过的预约才能签到";
    }
    
    // 检查签到状态
    if (!"NOT_CHECKED".equals(reservation.getCheckinStatus())) {
        if ("CHECKED_IN".equals(reservation.getCheckinStatus())) {
            return "该预约已签到，请勿重复操作";
        } else if ("CHECKED_OUT".equals(reservation.getCheckinStatus())) {
            return "该预约已完成签退，无法再次签到";
        } else {
            return "该预约签到状态异常";
        }
    }
    
    LocalDateTime now = LocalDateTime.now();
    
    // 检查是否在预约时间范围内（可提前15分钟签到）
    if (now.isBefore(reservation.getStartTime().minusMinutes(15))) {
        return "还未到签到时间，可提前15分钟签到";
    }
    
    if (now.isAfter(reservation.getEndTime())) {
        return "预约时间已结束，无法签到";
    }
    
    return null; // 验证通过
}
```

**签到签退规则：**
- **签到时间**: 预约开始前15分钟至开始后30分钟
- **签退时间**: 预约结束后30分钟内必须签退
- **验证机制**: 生成唯一验证码用于身份验证
- **超时处理**: 未按时签退自动记录违规

```java
public class CreditScoreCalculator {
    
    public static int calculateDeduction(String violationType) {
        switch (violationType) {
            case "NO_SHOW": return 10;      // 爽约未到扣10分
            case "OVER_TIME": return 5;     // 超时占用扣5分
            case "DAMAGE": return 20;       // 损坏设施扣20分
            case "MALICIOUS": return 15;    // 恶意预约扣15分
            case "NO_CHECKOUT": return 8;   // 未按时签退扣8分
            default: return 0;
        }
    }
    
    /**
     * 更新用户信用分
     */
    public void updateUserCreditScore(User user, String violationType) {
        int deduction = calculateDeduction(violationType);
        int newScore = user.getCreditScore() - deduction;
        user.setCreditScore(Math.max(newScore, 0));
        user.setViolationCount(user.getViolationCount() + 1);
        
        // 检查是否进入黑名单
        if (newScore < 60) {
            blacklistService.addToBlacklist(user.getId(), "信用分低于阈值");
        }
    }
}
```

```java
    if (ruleConfig == null) {

    long durationHours = Duration.between(reservation.getStartTime(), reservation.getEndTime()).toHours();
    if (ruleConfig.getMaxBookingHours() != null && durationHours > ruleConfig.getMaxBookingHours()) {

    }

    if (ruleConfig.getMinAdvanceBooking() != null) {
        long advanceHours = Duration.between(LocalDateTime.now(), reservation.getStartTime()).toHours();
        if (advanceHours < ruleConfig.getMinAdvanceBooking()) {

        }
    }
```
