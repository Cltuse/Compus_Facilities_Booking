# 实验室设备预约与维护管理系统

## 1. 系统功能介绍

实验室设备预约与维护管理系统是一个现代化的Web应用，旨在帮助高校和科研机构高效管理实验室设备的预约、使用和维护。

### 1.1 核心功能模块

#### 设备管理
- **设备信息管理**：完整的设备档案管理，包括设备名称、型号、规格、购买日期、价格等信息
- **设备分类管理**：支持设备按类别管理，可自定义分类和排序
- **设备状态跟踪**：实时监控设备状态（可用、使用中、维护中等）
- **设备搜索功能**：支持按名称、分类等多维度搜索和分页查询

#### 预约管理
- **在线预约**：用户可在线预约设备，选择使用时间段
- **预约审核**：管理员可审核预约申请，支持通过/拒绝操作
- **预约状态跟踪**：待审核、已通过、已拒绝、已完成、已取消等状态管理
- **预约历史**：用户和管理员可查看历史预约记录

#### 维护管理
- **维护记录**：记录设备的维护、保养、维修历史
- **维护跟踪**：跟踪维护进度和结果
- **设备关联**：维护记录与设备自动关联

#### 通知公告
- **公告发布**：管理员可发布系统通知和使用规范
- **公告管理**：支持公告的创建、编辑、发布和下线
- **用户通知**：重要信息实时推送给用户

#### 用户管理
- **角色权限**：支持管理员和普通用户两种角色
- **用户注册**：开放用户注册功能
- **个人信息**：用户可管理个人信息和修改密码
- **权限控制**：基于角色的访问控制，确保系统安全

#### 用户友好界面
- **欢迎页面**：用户登录后显示温馨的欢迎界面，包含每日箴言和励志内容
- **响应式设计**：适配桌面端和移动端设备
- **直观导航**：清晰的菜单结构和操作流程

## 2. 系统特色

### 2.1 技术特色
- **前后端分离**：采用现代化的前后端分离架构，提高开发效率
- **RESTful API**：标准化的API设计，便于扩展和维护
- **响应式UI**：基于Element Plus的现代化用户界面
- **单页应用**：流畅的用户体验，无需页面刷新
- **统一数据格式**：标准化的数据返回格式

### 2.2 业务特色
- **欢迎页面**：为用户登录提供温暖的使用体验，包含每日箴言和正能量内容
- **完整的预约流程**：从预约申请到审核完成的完整业务流程
- **灵活的设备管理**：支持设备分类、状态管理、多维度搜索
- **便捷的维护跟踪**：完整的设备维护生命周期管理
- **实时通知系统**：重要信息及时传达

### 2.3 安全特色
- **密码保护**：用户密码在API返回中自动隐藏
- **权限控制**：基于角色的功能访问控制
- **数据验证**：完整的输入数据验证机制
- **SQL注入防护**：使用JPA框架天然防护SQL注入

## 3. 技术栈

### 3.1 后端技术栈
- **核心框架**：Spring Boot 3.2.0
- **数据持久**：Spring Data JPA + Hibernate
- **数据库**：MySQL 8.0
- **构建工具**：Apache Maven 3.9
- **Java版本**：JDK 21
- **跨域支持**：Spring Boot CORS配置

## 5. 接口说明

### 5.1 统一响应格式

所有API接口均采用统一的响应格式：

```json
{
  "code": 200,           // 响应状态码
  "message": "success",  // 响应消息
  "data": {}            // 响应数据
}
```

## 6. 部署运行

### 6.1 环境准备

#### 6.1.1 数据库准备
```bash
# 启动MySQL服务
# 创建数据库
CREATE DATABASE lab_equipment_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入初始数据
mysql -u root -p lab_equipment_system < database.sql
```

#### 6.1.2 后端配置
数据库连接配置在 `backend/src/main/resources/application.properties`：
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/lab_equipment_system?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
spring.datasource.username=root   // 用户名，默认root
spring.datasource.password=123456   // 密码，默认123456
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.jpa.database-platform=org.hibernate.dialect.MySQL8Dialect

server.port=5681
```

### 6.2 启动后端服务

```bash
# 进入后端目录
cd backend

# 编译项目
mvn clean compile

# 启动服务
mvn spring-boot:run
```

后端服务启动成功后，将运行在 `http://localhost:5681`

### 6.3 启动前端服务

```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

前端服务启动成功后，将运行在 `http://localhost:3000`
